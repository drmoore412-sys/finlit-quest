# Testing FinLit Quest

Run the complete dependency-free test suite from the project root:

```sh
./scripts/test.sh
```

The suite uses Node's built-in test runner. It covers content validation, stable references, world loading inputs, duplicate letters, word classification, crossing-safe hints, legacy and version-2 migration, malformed saves, sparse progress, all four review outcomes, interval progression, due dates, separate puzzle/challenge metrics, mastery limits, objective approval and relationships, objective progress, future scenario/challenge hooks, review counts, and unlock behavior.
